# MCOD: The First Challenging Benchmark for Multispectral Camouflaged Object Detection [ACM MM 2025]
## Paper
Paper is avaliable at [arXiv](https://arxiv.org/abs/2509.15753) and [ACM MM 2025](https://dl.acm.org/doi/abs/10.1145/3746027.3758294).

| Sensor | Start WL (nm) | Stop WL (nm) | CWL (nm) | FWHM (nm) | Colour   |

| S2     | 455           | 520          | 487.5    | 65        | Blue     |
| S3     | 525           | 575          | 550      | 50        | Green    |
| S4     | 580           | 625          | 602.5    | 45        | Orange   |
| S5     | 630           | 690          | 660      | 60        | Red      |
| S6     | 705           | 745          | 725      | 40        | Red Edge |
| S7     | 750           | 820          | 785      | 70        | NIR 1    |
| S8     | 825           | 950          | 887.5    | 125       | NIR 2    |
